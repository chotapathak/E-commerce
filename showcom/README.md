
# Shopping State

<h1 align='center'>

<p align='center'>

</p>

</h1>
First go to showcom folder
Second install dependencies
third, run the development server:

```bash
cd E-commerce/showcom
npm i
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
